# Task 1.9: Add DNS Override for Proxmox (FQDN)

**Objective**: Create DNS entries so both `proxmox` and `numenor` (and their FQDNs) resolve to `192.168.1.10`.

**Steps**:
1. In OPNsense: **Services → Unbound DNS → Overrides**
2. Create first override (themed name):
   - Click **Add** button
   - Configure:
     - **Host**: `numenor`
     - **Domain**: `cuffney.com`
     - **Type**: `A`
     - **Value**: `192.168.1.10`
     - **Description**: `Proxmox host (numenor - themed)`
   - Click **Save**
3. Create second override (descriptive name):
   - Click **Add** button
   - Configure:
     - **Host**: `proxmox`
     - **Domain**: `cuffney.com`
     - **Type**: `A`
     - **Value**: `192.168.1.10`
     - **Description**: `Proxmox host (proxmox - descriptive)`
   - Click **Save**
4. Create third override (short themed name, no domain):
   - Click **Add** button
   - Configure:
     - **Host**: `numenor`
     - **Domain**: (leave empty)
     - **Type**: `A`
     - **Value**: `192.168.1.10`
     - **Description**: `Proxmox host (numenor - short)`
   - Click **Save**
5. Create fourth override (short descriptive name, no domain):
   - Click **Add** button
   - Configure:
     - **Host**: `proxmox`
     - **Domain**: (leave empty)
     - **Type**: `A`
     - **Value**: `192.168.1.10`
     - **Description**: `Proxmox host (proxmox - short)`
   - Click **Save**
6. Click **Apply Changes**

**Test**:
```bash
# Test DNS resolution (all variations should work)
nslookup numenor
# Should resolve to: 192.168.1.10

nslookup proxmox
# Should resolve to: 192.168.1.10

nslookup numenor.cuffney.com
# Should resolve to: 192.168.1.10

nslookup proxmox.cuffney.com
# Should resolve to: 192.168.1.10

dig @192.168.1.1 numenor.cuffney.com
# Should return: numenor.cuffney.com.  IN A 192.168.1.10

dig @192.168.1.1 proxmox.cuffney.com
# Should return: proxmox.cuffney.com.  IN A 192.168.1.10
```

**Expected Result**: Both `numenor.cuffney.com` and `proxmox.cuffney.com` (and short names) resolve to `192.168.1.10`.
